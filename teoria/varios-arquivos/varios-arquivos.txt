arquivos.h: usa para variáveis globais, protótipos de função e definição de novos tipos. 
novas structs voce coloca nesse .h também

incorpora ao arquivo.c usando o #include

arquivos.c: implementação dos protótipos de função que estiverem no arquivo.h. nesse ponto C vc 
tbm precisa dar um #include no aquivo.h para deixar relacionado tgldo. 
vc faz #include "arquivo.h" -- precisa dessa aspas duplas mesmo. no main você coloca com as aspas duplas também

para quando for rodar no gcc vc precisa rodar todos os .c atrelados. Estando em uma pasta certinha você pode fazer como eu fiz:
gcc *.c -o ./exercicio-1D.x -Wall -Wextra
note que só gera um executável, mesmo que sejam vários .c
no meu caso se eu não usasse esse atalho do linux (o *), eu teria que fazer:
gcc exercicio-1D.c funcoes-criadas.c -o ./exercicio-1D.x -Wall -Wextra
